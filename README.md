# BrainBuddy – AI Chat Assistant

BrainBuddy is a modern AI-powered chatbot built with React.js and OpenRouter API integration. It enables users to interact with Large Language Models through a clean, responsive, and intuitive chat interface.

## Features

* Real-time AI-powered conversations
* Responsive and modern user interface
* Secure API key management using environment variables
* Dynamic message rendering
* Fast and lightweight frontend built with Vite
* Reusable React component architecture

## Tech Stack
* React.js
* JavaScript
* Vite
* OpenRouter API
* CSS

##  Installation
```bash
git clone https://github.com/khushigupta-kg/BrainBuddy.git
cd BrainBuddy
npm install
```

##  Run Locally
```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```

## How It Works
1. User enters a prompt.
2. React sends the request to the OpenRouter API.
3. The selected language model processes the request.
4. The response is displayed in the chat interface.
